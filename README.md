# ServerPython
Código para atividade de pontuação para a AV2 da matéria de Redes de Computadores, do curso de Ciência da Computação.

Este trabalho apresenta a criação de um Servidor Web e suas
principais características, além de explorar as funcionalidades do
Wireshark. Os conteúdos fazem parte da disciplina de Redes de
Computadores, ministrada pelo professor José Raul de Brito. O material foi
desenvolvido de forma clara e objetiva, garantindo coesão e
coerência na abordagem dos temas.
